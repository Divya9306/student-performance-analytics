# Student Performance Analytics System

## Project Overview

A full-stack application built to learn:

- Database Management Systems
- MySQL
- SQL Queries
- Node.js
- Express.js
- Frontend Development

## Features

- Student Management
- Marks Management
- Attendance Tracking
- Analytics Dashboard

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MySQL

- CRUD Operations
- Joins
- Subqueries
- Views
- Stored Procedures
- Triggers
- Indexing
- Transactions
